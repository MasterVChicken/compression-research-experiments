# compression-research-experiments

Reproducible experiment scripts and configs for our work on **lossy
compression of scientific data** (zfp, MGARD, SZ, … on SDRBENCH datasets).

Scripts and configs only — no datasets or compressed binaries are tracked here.

## Structure

One directory per paper. Each is self-contained and has its own `README.md`
with the exact commands to reproduce that paper's results.
